## Question Answer Application

- Filtering Posted Questions
- Notification System
- Label System
- Recording a Question
- Question Search
- Correct Answer System
- Answer Scoring System

## Setup
- `php composer.phar install` or `composer install`
- `php artisan migrate`
- `php artisan cache:clear`
